FloatingActionButton详解

发表于 2017-05-02 | 分类于 Android

FloatingActionButton是MD中的控件，首先在build.gradle中添加对design的支持包：

FloatingActionButton的扩展属性有：
app:borderWidth=””———-边框宽度，通常设置为0 ，用于解决Android 5.X设备上阴影无法正常显示的问题。
app:layout_anchor=””———-指定参照物。
app:layout_anchorGravity=””———-指定相对于参照物的位置。
app:layout_behavior=””———-设置behavior。
app:rippleColor=””———-点击的边缘阴影颜色。
app:backgroundTint=””———-按钮的背景颜色，不设置，默认使用theme中colorAccent的颜色。
app:elevation=””———-边缘阴影的宽度。
app:pressedTranslationZ=””———-点击按钮时，按钮边缘阴影的宽度，通常设置比elevation的数值大。
FloatingActionButton间接继承ImageButton和ImageView当然就可以把它当做这两个View来使用也是可以的。
xml布局代码如下：

<android.support.design.widget.CoordinatorLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    android:layout_width="match_parent"
    android:layout_height="match_parent">
    <ImageView
        android:id="@+id/image"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:scaleType="fitXY"
        android:src="@mipmap/splash01"/>

    <android.support.design.widget.FloatingActionButton
        android:id="@+id/fl_button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        app:layout_anchor="@+id/image"
        app:layout_anchorGravity="right|bottom"
        android:layout_margin="10dp"
        android:scaleType="fitXY"
        android:src="@mipmap/ic_add"/>
</android.support.design.widget.CoordinatorLayout>
