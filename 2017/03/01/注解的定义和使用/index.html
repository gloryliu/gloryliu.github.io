注解的定义和使用

发表于 2017-03-01 | 分类于 开发经验

java注解的使用步骤：
1、声明注解：

注解是通过基于接口机制创建的，关键字interface前面的@符号告诉编译器，这是一种注解类型。
注解声明没有方法体，所有的注解都自动扩展Annotation。
2、指定保留策略：

保留策略是通过java内置注解@Retention实现的。三种保留策略分别为:

SOURCE：只在源文件中保留，在编译的时候会被抛弃。
CLASS：在编辑时被存储在.class文件中，在运行时JVM得不到这些注解。
RUNTIME：在编译时被存储在.class文件中，在运行时JVM可以获取到这些注释，RUNTIME保留策略提供了最永久的注解。
