centOs安装Python3.5.3

发表于 2017-04-26 | 分类于 python

安装前期准备

  • 1、安装gcc
    yum install gcc
  • 2、安装wget
    yum install wget
  • 3、安装开发工具包
    yum groupinstall “Development tools” -y
  • 4、安装openssl和openssl-devel
    yum install openssl openssl-devel -y

  • 5、安装readline-devel
    yum install readline-devel -y

    安装

    1、下载 

    wget https://www.python.org/ftp/python/3.5.3/Python-3.5.3.tgz

    2、解压进入目录 

    [root@client0 ~]#
tar -zxvf Python-3.5.3.tgz
[root@client0 ~]#
cd Python-3.5.3

    3、配置环境

    查看有哪里参数
    [root@client0 Python-3.5.3]#
    ./configure -h
    配置安装目录，我这里只配置的安装目录
    [root@client0 Python-3.5.3]#
    ./configure –prefix=/usr/local/Python3.5.3
    4、安装

    make && make install

5、创建链接文件 

[root@client0 Python-3.5.3]# ln -s /usr/local/Python3.5.3/bin/python3.5 /usr/bin/python3.5
[root@client0 bin]# ln -s /usr/local/Python3.5.3/bin/pip3 /usr/bin/pip

使用的时候就用python3.5，系统原装的最好不要动不然会有很多麻烦，比如yum。
